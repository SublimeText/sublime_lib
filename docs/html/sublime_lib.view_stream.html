sublime_lib.view_stream module¶

class sublime_lib.view_stream.ViewStream(view, *, force_writes=False)¶

A TextIOBase encapsulating a View object.

All public methods (except flush()) require that the underlying View object be valid (using View.is_valid()). Otherwise, ValueError will be raised.

The read(), readline(), write(), and tell() methods require that the underlying View have exactly one selection, and that the selection is empty (i.e. a simple cursor). Otherwise, ValueError will be raised.

read(size)¶

Read and return at most size characters from the stream as a single str. If size is negative or None, reads until EOF.

readline(size=-1)¶

Read until newline or EOF and return a single str. If the stream is already at EOF, an empty string is returned.

write(s)¶

Insert the string s into the view and return the number of characters inserted. The string will be inserted immediately before the cursor.

Note: Because Sublime may convert tabs to spaces, the number of characters inserted may not match the length of the argument.

flush()¶

Do nothing. (The stream is not buffered.)

seek(offset, whence=0)¶

Move the cursor in the view to the given offset. If whence is provided, the behavior is the same as for TextIOBase. If the view had multiple selections, none will be preserved.

seek_start()¶

Move the cursor in the view to before the first character.

seek_end()¶

Move the cursor in the view to after the last character.

tell()¶

Return the character offset of the cursor.

clear()¶

Erase all text in the view.
