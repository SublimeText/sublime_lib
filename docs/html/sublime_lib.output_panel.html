sublime_lib.output_panel module¶

class sublime_lib.output_panel.OutputPanel(window, name, *, force_writes=False)¶

A ViewStream wrapping an output panel in the given window with the given name.

Raises

ValueError – if window has no output panel called name.

classmethod create(window, name, *, force_writes=False, unlisted=False, **kwargs)¶

Create a new output panel with the given name in the given window.

If kwargs are given, they will be interpreted as for new_view().

full_name¶

The output panel name, beginning with output.. Generally, API methods specific to output panels will use name, while methods that work with any panels will use full_name.

is_visible()¶

Returns True if the output panel is currently visible.

show()¶

Shows the output panel, hiding any other visible panel.

hide()¶

Hides the output panel.

toggle_visibility()¶

If the output panel is visible, hide it; otherwise, show it.

destroy()¶

Destroys the output panel.
