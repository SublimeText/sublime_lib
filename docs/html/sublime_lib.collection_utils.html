sublime_lib.collection_utils module¶

sublime_lib.collection_utils.projection(d, keys)¶

Return a new dict with keys restricted to values in keys.

>>> projection({'a': 1, 'b': 2}, ['b'])
{'b': 2}

If keys is a dict, then it maps keys of the original dict to keys of the result:

>>> projection({'a': 1, 'b': 2}, {'b': 'c'})
{'c': 2}
